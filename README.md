# SafeNote Admin Hub

Upload these files to your Render SafeNote project:

- `admin_hub.html`
- `safenote_api.py`

Keep your existing:
- `safenote_sa.html`
- `nhw_portal.html`
- `requirements.txt`

Then redeploy Render and open:

`https://YOURDOMAIN/admin`

Use your Render `ADMIN_SECRET` to unlock the hub.

The Admin Hub lets you:
- create NHW/CPF codes
- copy/revoke codes
- view reports
- filter reports
- export CSVs
- clear test data
